طريقة الرفع:
1) فك الضغط.
2) ارفع كل الملفات مباشرة في جذر مستودع GitHub.
3) تأكد أن index.html موجود في root وليس داخل مجلد فرعي.
4) فعّل GitHub Pages من:
   Settings > Pages > Deploy from branch > main > /(root)

ملاحظات مهمة:
- الموقع مرتبط مباشرة مع Apps Script API الذي أرسلته.
- لوحة الأدمن تعتمد على أن المستخدم المسجل دخوله دوره admin في Sheet users.
- هذه نسخة Static Frontend، لذلك الحماية الحقيقية ما زالت محدودة لأن الـ API عام.
- الأفضل لاحقًا إضافة طبقة تحقق للأدمن داخل Apps Script.
