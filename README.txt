عقاري - النسخة النظيفة V3

طريقة الرفع:
1) فك الضغط.
2) ارفع كل الملفات مباشرة داخل جذر المستودع في GitHub.
3) لا تضع الملفات داخل مجلد فرعي.
4) فعّل GitHub Pages من:
   Settings > Pages > Deploy from branch > main > /(root)

ملاحظات:
- تسجيل المستخدمين يعتمد على: phone + password
- تسجيل الدخول يعتمد على: phone + password
- لوحة الأدمن تعتمد على أن الحساب المسجل دخوله role=admin
- يوجد فلترة وعرض للعقارات المعتمدة
- يوجد إدارة Pending / Approved / Rejected
